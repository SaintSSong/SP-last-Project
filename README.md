## 패키지 설치

```bash
$ yarn install
```

## 실행 방법

```bash
# 서버 실행(배포)
$ yarn run start

# 서버 실행(개발)
$ yarn run start:dev

```

## Test

```bash
# 테스트 실행
$ yarn run test

# 테스트 커버리지 확인
$ yarn run test:cov
```
